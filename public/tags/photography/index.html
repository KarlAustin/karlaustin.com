cPanel - Broken FTPS (FTP over TLS) with Java Clients

May 9, 2016

2 minute read

If you’ve gone to edit your site and suddenly found that your chosen IDE won’t connect to upload any more via FTPS (FTP using TLS encryption) and your web host runs cPanel, then chances are they’re running v56 and your IDE is written in Java. I hit this issue with another website late last week. Your IDE probably just says it can’t connect, that’s what JetBrains WebStorm and PhpStorm do although the logs did give more information: Caused by: java.security.InvalidAlgorithmParameterException: Prime size must be multiple of 64, and can only range from 512 to 2048 (inclusive) NetBeans gave the game away sooner by giving the error from Java about DH parameters.
